# Scenario 05: Creative Expression

category: creative_expression

## User Profile

A professional wants to start publishing short videos about workplace thinking but is afraid the content will be boring.

## Initial Input

"我想学习做短视频表达，把我工作里的思考讲出来，但我怕自己讲得很无聊。"

## Expected Skill Chain

1. `goal-clarification`
2. `level-assessment`
3. `learning-map`
4. `path-planning`
5. `resource-curation`
6. `tutoring-coach`
7. `review-adjustment`

## Acceptance Focus

- Define target as publishing a small number of videos, not mastering all video production.
- Assess topic clarity, speaking comfort, script ability, editing tools, and audience.
- Plan should include scripts, rough recordings, feedback, and publishing cadence.
- Coach should review actual scripts or spoken drafts.

## Failure Signals

- Over-focuses on equipment and editing software.
- No publishing output.
- Gives motivational advice without critique.
