# Skill: Level Assessment

Use this skill when the learning goal exists but the learner's current position is uncertain.

## Purpose

Locate the learner's current level with enough evidence to generate a useful learning path.

## When To Use

- A goal contract exists but baseline ability is unknown.
- The learner self-rates without evidence.
- The domain requires prerequisites.
- A plan would be irresponsible without knowing current level.

## Assessment Modes

Choose the lightest mode that provides enough signal.

| Mode | Best For | Example |
| --- | --- | --- |
| Self-audit | Low-risk goals and quick onboarding. | Ask for prior experience, recent attempts, confidence. |
| Diagnostic Q&A | Knowledge understanding and career growth. | Ask conceptual and situational questions. |
| Mini task | Skill training and project delivery. | Ask learner to write, solve, design, speak, or build a small sample. |
| Artifact review | Creative, career, project, or writing goals. | Review a past work sample. |
| Mock test | Exam certification. | Give representative questions or interpret a score report. |
| Routine audit | Life practice. | Review current routine, constraints, triggers, and safety issues. |

## Process

1. Select assessment mode based on goal category.
2. Explain what evidence you need and why.
3. Ask for existing evidence first; do not create a test if the learner already has useful work samples.
4. Run a short assessment: usually 3 to 7 questions or one small task.
5. Identify strengths, weaknesses, prerequisites, blockers, and confidence.
6. If confidence is low, mark missing evidence rather than overclaiming.

## Output Format

```markdown
## Current Position

- Estimated level:
- Confidence:
- Evidence used:
- Strengths:
- Weaknesses:
- Missing prerequisites:
- Key gap to target:
- Recommended next assessment, if needed:
```

## Quality Bar

A good assessment is:

- Evidence-backed.
- Proportionate to the goal.
- Clear about confidence.
- Useful for planning the next step.
- Not humiliating or overly clinical.
