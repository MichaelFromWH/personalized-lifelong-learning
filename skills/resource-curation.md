# Skill: Resource Curation

Use this skill when the learner needs materials, examples, exercises, datasets, references, or practice sources.

## Purpose

Curate resources that support the current learning path, not a generic list of popular materials.

## When To Use

- A task requires external material.
- The learner asks what to read, watch, practice, or use.
- The current material is too easy, too hard, outdated, boring, or misaligned.
- A plan needs exercises or authentic examples.

## Process

1. Identify the exact route node or task the resource supports.
2. Choose resource types: primer, reference, exercise, project example, dataset, community, mentor feedback, or assessment.
3. Prefer a small number of high-signal resources.
4. Explain how to use each resource.
5. Add a stop rule so the learner does not over-consume.
6. If current web lookup is required, use available browsing or search tools and cite sources.

## Output Format

```markdown
## Resource Shortlist

| Resource | Type | Supports | How To Use | Stop Rule |
| --- | --- | --- | --- | --- |

## Why These Resources

- 
```

## Quality Bar

A good resource recommendation reduces friction and increases progress. It should not become a substitute for practice.
