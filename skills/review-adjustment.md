# Skill: Review Adjustment

Use this skill at review points or when progress evidence suggests the plan should change.

## Purpose

Compare evidence against the goal contract, update the learning state, and decide whether to continue, adjust, upgrade, pause, or graduate.

## When To Use

- A scheduled review date arrives.
- The learner completes a stage.
- The learner repeatedly misses tasks.
- The learner progresses faster than expected.
- New evidence contradicts the old assessment.
- The learner asks whether they have achieved the goal.

## Process

1. Gather evidence since the last review.
2. Compare it with the current plan and goal contract.
3. Identify what changed: ability, motivation, constraints, preference, or goal.
4. Update level and confidence.
5. Decide:
   - continue current plan
   - adjust workload
   - change learning path
   - change guidance style
   - upgrade target
   - pause
   - graduate
6. Produce the next plan or next review rule.

## Output Format

```markdown
## Review Result

- Evidence reviewed:
- Progress judgment:
- Updated level:
- Confidence:
- What worked:
- What did not work:
- Decision:
- Plan adjustment:
- Next review date:
```

## Quality Bar

A good review is honest without being discouraging. It should change the plan when evidence demands it and preserve momentum when the learner is on track.
