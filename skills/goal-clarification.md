# Skill: Goal Clarification

Use this skill when the learner's goal is vague, too broad, unbounded, or missing success criteria.

## Purpose

Turn a learning wish into a goal contract the learner can actually navigate toward.

## When To Use

- The learner says "I want to learn X" without a concrete outcome.
- The learner has an abstract improvement area but does not know what level to target.
- The learner wants help choosing a goal level.
- The learner has not selected a guidance style.

## Inputs

- Learner's initial wish.
- Known background and constraints.
- Available time, deadline, and preferred intensity if known.
- Any prior learner profile.

## Process

1. Restate the learner's wish in plain language.
2. Classify the goal category and mention the classification only if useful.
3. If the target is vague, offer 3 to 5 concrete target levels.
4. Ask one question at a time until the destination is concrete.
5. Help the learner choose a guidance style in learner-facing language:
   - Relaxed companion
   - Direct mentor
   - Question-led coach
   - Action coach
   - Challenge mode
6. Convert the result into a goal contract.
7. Break the clarified goal into an OKR: one Objective, 2 to 4 Key Results, and evidence for each Key Result.

## Target Level Ladder

When a domain lacks a formal rating system, use this generic ladder:

| Level | Meaning | Evidence |
| --- | --- | --- |
| L1 Awareness | Knows basic terms and can follow beginner explanations. | Can summarize core concepts. |
| L2 Working Understanding | Can apply concepts in simple cases. | Can solve guided examples. |
| L3 Independent Use | Can perform real tasks without step-by-step help. | Can finish a small authentic task. |
| L4 Fluent Performance | Can handle varied cases and explain tradeoffs. | Can complete realistic work with quality. |
| L5 Creative / Professional | Can create, teach, critique, or innovate in the field. | Can produce high-quality original work. |

## Output Format

```markdown

- Goal category:
- Learning goal:

- Target level:

- Constraints:

- Open uncertainty:
```

## Quality Bar

A good goal contract answers:

- What exactly is the learner trying to do?
- What counts as success?
- Which Key Results prove the goal is moving?
- What time and energy can they realistically invest?
- What type of evidence will prove progress?
- How should the agent guide them?

If these are not clear, keep clarifying before planning.
